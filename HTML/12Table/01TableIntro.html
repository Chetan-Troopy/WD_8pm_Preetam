Table In HTML

  • table => table tag is used to make table in HTML
  • thead => thead section is top part of table in which we write headings
  • tbody => table body section
  • tfoot => table footer section
  • tr => table row - to create rows in table
  • th => table heading -> In this text will be in center and Bold
  • td => table data
